Remote Sensing Monitoring System

System Overview:

RRG will use a multi-prong approach to track, validate, and monitor planting activities. RRG will leverage geospatial tools to ensure planting claims are accurate, auditable, and traceable. RRG will build the monitoring capacity of project partners and provide the foundational data needed to ensure transparency, traceability, and permanence

System Goals:

  • Track and validate planting activities throughout the project area
  • Assess forest structure development via multiple imagery sources
  • Monitor site conditions (precipitation, topography, hydrology, soil maps)
  • Monitor mortality over time
  • Evaluate social outcomes (jobs, engagement, training)
  • Support VCU-related metrics: carbon sequestration estimates, leakage and performance benchmark

Commercial Satellite Imagery

RRG will utilize commercial imagery to provide up-to-date imagery for planting validation and monitoring. RRG will purchase commercial imagery bi-yearly to monitor planting and mortality.

Currently, RRG has partnered with Apollo Image Hunters, who provide satellite imagery from Maxar Technologies, Airbus and Jilin. Apollo provides two images on purchase. Below is the description of each image and their function in project monitoring.

  • Three-Band (RGB) High Resolution Imagery (50cm)
    • Planting validation including acres planted metrics
    • Density (trees/hectare) once seedlings are established
    • Mortality monitoring
    • Carbon sequestration monitoring utilizing fixed-radius plots to measure changes in canopy cover
  • Four-Band (RGB, NIR) High Resolution Imagery (50cm)
    • Calculate Normalized Difference Vegetation Index (NDVI)
    • Planting validation including planting row verification
    • Biomass mapping

Public Satellite Imagery

RRG will utilize public imagery to provide up-to-date imagery for planting validation, biomass mapping and monitoring. RRG will update data quarterly to monitor planting and mortality.

RRG utilizes Google Earth Engine (GEE), which provides access to an abundance of satellite imagery and geospatial data.
Geospatial data provides RRG with the ability to

  • Validate planting activities and hectares restored
  • Monitor biomass development utilizing stocking indices (SI) and statistical learning
  • Monitor site performance and mortality overtime

Each GEE datasource will be used as a SI index. SI is an unspecified remote sensing metric that has demonstrated correlation with terrestrial aboveground carbon stocks. Below is the description of each data source that will be used in project monitoring.

  • NASA Landsat-8 Satellite Program
    • Hyperspectral Imagery
    • Normalized Difference Vegetation Index (NDVI)
    • Normalized Difference Fraction Index (NDFI - Best in jungle environments)
    • GEE datasource
  • JAXA ALOS PALSAR Satellite Program
    • Synthetic Aperture Radar (SAR) Imagery
    • HH and HV Backsplatter
    • GEE datasource
  • NASA GEDI Satellite Program
    • LiDAR Imagery
    • Canopy Height
    • GEE datasource
