Project Forest: Measuring

Interact with RRG monitoring and measuring systems.

Project Area Maps

RRG has collaborated with the Friends of the Usambara to accurately map project boundaries. In total, RRG and FOU have outlined ~11,000 hectares for the planting of 16,490,000 trees.

Forest Inventories

RRG traveled to Tanzania and conducted inital forest inventories of planted species. Data collected included tree species, diameter at breast height (DBH) and age. RRG uses this data for ex-ante estimates of biomass removals.

Ex-Ante Estimates of Biomass Removals

RRG is able to forecast a range of biomass removal performance scenarios utilizing LiDAR biomass data for the Usambara and RRG's inventory data. Click here for RRG's technical modeling document.

Observed Biomass Changes Using Geospatial Tools

RRG will utilize Synthetic Aperature Radar (SAR) data provided by the JAXA ALOS PALSAR Satellite Program. SAR data has a proven coorelation with biomass. With regular updates in image availability, RRG is able to track biomass removals in real time.Click here for RRG's technical modeling document.
Much of Project Forest's planting has just begun. RRG expects biomass estimates to be low due to planting prep and small seedling size (0.5-1m).
