Project Forest: Verification

Field Maps Verification Program

RRG is using a multiprong system of remote sensing technologies to validate project activities including RGB imagery, biomass mapping and in-field data recording. For aerial RGB imagery, RRG is utilizing commercial products provided by the Jilin satellite program. For biomass mapping, RRG is utilizing products provided by the JAXA satellite program. Using these systems, RRG is able to track and validate FOU's claims of hectares planted. Click here for a tutorial on how RRG is validating project activities.

** Commercial imagery is only 50cm resolution. RRG is currently not able to validate individual trees outside of partner collected data. Seedlings are anticipated to be 0.5-1m in height. As the stands mature, RRG will be able to see and validate planting densities per hectare.
** Due to imagery availability, RRG is only able to validate planting activities through 2/25/2025.
** RRG is slowly rolling out an in-field tree verification system. Expect increased verified tree metrics late 2025.

Project Malawi: Verification

Remote Sensing Verification Program

RRG is using a multiprong system of remote sensing technologies to validate project activities including RGB imagery and biomass mapping. MDOF needed to clear the project sites of vegetation prior to project activities, resulting in a barren landscape. Yet, RRG is able to verify MDOF activity via planting rows. As the stand matures, RRG will gather more data including densities, biomass and mortality events.
