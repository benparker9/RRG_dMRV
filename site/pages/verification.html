Project Forest: Verification

Interact with internal audits and 3rd party verifications.

** Commercial imagery is only 50cm resoltution. Due to this, RRG is currently not able to validate individual trees. Seedlings are anticipated to be 0.5-1m in height. As the stands mature, RRG will be able to validate individual trees.
** Due to imagery availability, RRG is only able to validate planting activities through 2/25/2025.
** RRG is currently able to validate planted areas utilzing RGB, NDVI and SAR imagery. Using partner-reported data, RRG computes estimated planting densities to compare to the original anticipated 1,600 trees per hectare.
** RRG is slowly rolling out an in-field tree verification system. Expect verified tree metrics late 2025.
